# Ludarte Dashboard

O `Ludarte Dashboard` organiza a documentacao do projeto em uma estrutura clara de arquitetura, frontend, backend e banco de dados.

## Objetivo

Centralizar a visao do produto e registrar decisoes tecnicas de implementacao.

## Navegacao

- Backend: responsabilidades, contratos e integracoes.
- Database: modelagem e estrategia de persistencia.
- Frontend: interface, fluxos e experiencia visual.
